Change Log

v2.0.0

  • Revived the site.
  • Dropped unaccessible links to faction analyzer.
  • Upgrade to latest libraries and fixed 46 vulnerabilities (17 low, 11 moderate, 18 high).

v1.3.0

  • Fixed the website not working on mobile and non-chrome browsers. (issue #29)
  • Fixed the 'Hamburger' icon (drop down) not appearing on the top navigation bar on mobile. (issue #31)
  • Fixed a typo for Fav12 in the detailed game stats table. (issue #15)
  • Fixed Fire and Ice bonuses not appearing in the detailed game stats table. (issue #10)
  • Improved support for the different versions of the variable factions. (issue #7)

v1.2.0

  • Added a branch new page for showing faction by faction statistics
  • These faction statistics are based off of League game results. I included all games from Divisions 1-3 in Seasons 1-10. I'll add newer seasons soon.

v1.1.2

  • Fixed taking 2 duplicate town tiles simultaneously (like '+2tw3') not parsing correctly. (issue #11)
  • Fixed parsing the short form of leeching/declining ('leech 2' opposed to 'leech 2 from dwarves') not parsing correctly. (issue #12)
  • Fixed support for Unicode characters in player names not parsing correctly. (issue #13)
  • Fixed changing the value of the search box renaming the game name on the game analysis page. (issue #14)

v1.1.1

  • Fixed a bug causing a category to not display in the faction graph if the first faction in that category didn't score points for that category

v1.1.0

  • Changed the faction score graph to be horizontal.
  • Improved mobile support for the horizontal faction score graph (above)
  • Game name appears in search box when hardlinking to a page.

v1.0.4

  • Reworked handling for leeching to better handle Cultists and Shapeshifters. (They can get +CULT or Gain P3 Token on the same line as the leech, which was messing stuff up)
  • Fixed Cultist SH not counting correctly after the 1.0.3 update

v1.0.3

  • Dragonlords, Acolytes, and Riverwalkers no longer get round points for spades generated in ROUND1 (Spd >> 2, 1Earth -> c)
  • Chaos Magicians SH Action now correctly allows two actions in one turn (this required a fair amount of rework)

v1.0.2

  • Added unknown points to the total score (I still want to fix these as I find them, but don't want them to throw off other values)
  • Fixed capitalization issues in 'advance ship' and 'advance dig'
  • Riverwalkers 'unlock-terrain xxx' action is now being parsed

v1.0.1

  • Added the change log
  • Fixed the site from always loading the game 'onion'
